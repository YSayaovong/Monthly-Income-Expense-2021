
# Nonprofit Monthly Income & Expense Financial Model (2021)

This project is a complete 12-month financial reporting and budgeting model for a nonprofit organization. It includes seasonally-adjusted Actual/Budget figures, automated variance calculations, and a consolidated year-end financial summary. The structure is modeled after real-world treasurer responsibilities, and all data has been fully anonymized and replaced with realistic mock figures.
